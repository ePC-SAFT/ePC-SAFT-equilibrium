# Private Water Self-Ionization Provider Integration

Status: implemented private non-production consistency checkpoint

Authority effect: none

Depends on: D-028 private homogeneous reacting-phase foundation

## Outcome

Add the first source-complete nonideal reacting-liquid input for the private
Equilibrium kernel: water self-ionization at `298.15 K` and `100000 Pa`, using
one exact installed Provider artifact and the reaction

```text
2 H2O <-> H3O+ + OH-
```

The slice proves that a source equilibrium constant with mixed standard states
can be transformed into the Provider Helmholtz coordinate basis and solved with
the existing conservation, electroneutrality, Ipopt, and certificate owners.
It adds no public operation, result type, selector, release, promotion,
predictive claim, or authority change. D-026 remains the public HELD2 admission
gate.

## Source identities and limits

The ePC-SAFT model is the original aqueous Debye--Huckel formulation and
parameter set reported by Held, Cameretti, and Sadowski, *Fluid Phase
Equilibria* 270 (2008) 87--96, DOI
`10.1016/j.fluid.2008.06.010`. The retained source defines water with the 2B
association model, treats ionic species with the Debye--Huckel contribution,
omits ion--ion dispersion, and reports the hydrated `H+` and `OH-` parameters
at `25 degC`. Because the paper states that aqueous `H+` exists as hydronium,
the true Provider species identifier is `hydronium-cation` with formula
`H3O+`, not a bare-proton alias.

The reaction constant is evaluated from IAPWS R11-24, *Revised Release on the
Ionization Constant of H2O* (2024). That release defines the reaction above,
uses molal standard states for the ionic species and a mole-fraction standard
state for water, and supplies `pKw(T, rho_w)`. The retained scalar input for
this slice is evaluated at `298.15 K` and the ordinary-water density belonging
to `100000 Pa`; the evaluation record retains the equation revision, constants,
units, input density source, and computed dimensionless `ln Kw`.

Held parameters and IAPWS `Kw` serve different roles and remain separately
identified. The Provider model is not claimed to reproduce IAPWS water density,
and `Kw` is not evaluated from the Provider-predicted density.

## Provider-owned model and SDK tail

Provider adds one immutable installed bundle with ordered components

```text
(water, hydronium-cation, hydroxide-anion)
```

and charges `(0, +1, -1)`. The bundle contains only source-backed component,
water-correlation, association, pair, model, and domain records. Its topology
is `ionic-held-2008-debye-huckel`. It reuses Provider's existing hard-chain,
dispersion, association, Debye--Huckel, density, packing, and CppAD owners.
Born and SSM+DS contributions are exactly absent for this topology; current
Figiel or Ascani parameters are never mixed into it.

The version-1 native SDK receives an append-only tail containing:

- a stable Helmholtz basis identifier;
- `reference_amount_mol = 1` and
  `reference_number_density_mol_per_m3 = 1`;
- resolved source pressure bounds, represented as `NaN` only when the source
  declares no pressure interval; and
- one fixed-`T,P`, one-solvent, one-cation/one-anion standard-reference
  callback.

The callback follows the existing exactly electroneutral formula-unit dilution
sequence and returns only quantities with a neutral thermodynamic meaning:

- the sum of the cation and anion infinite-dilution log fugacity coefficients;
- the pure-solvent log fugacity coefficient;
- the solvent molar mass;
- the accepted terminal reference molality and convergence error;
- the exact pure-solvent liquid molar volume selected by Provider's existing
  fixed-pressure density closure for direct-solve initialization; and
- the model fingerprint, component order, charge order, state, and basis ID.

It exposes no reaction, equilibrium constant, standard chemical potential,
single-ion predictive claim, solver, or chemistry registry. All values derive
from the existing Provider state/reference owners.

## Exact standard-state transformation

Let `q_i` be the dimensionless mechanical chemical potential generated by the
Provider phase block, and let the Provider ideal concentration reference be
`c_ref = 1 mol m^-3`. At fixed pressure,

```text
q_i = ln(x_i P / (R T c_ref)) + ln(phi_i).
```

For the 1:1 ionic formula-unit path in pure water, define the gauge-invariant
mixed-standard offset

```text
Delta C = 2 ln(M_w m_standard)
        + [ln(phi_H3O_inf) + ln(phi_OH_inf)]
        - 2 ln(phi_water_pure),
```

where `M_w` is in `kg mol^-1` and `m_standard = 1 mol kg^-1`. The fixed-pressure
ideal concentration factors cancel because the reaction has equal total
stoichiometric amount on both sides.

Equilibrium constructs

```text
lnK_provider = lnKw_IAPWS + Delta C
```

and passes that value to the existing compiler, which constructs `g_ref` from

```text
nu g_ref = -lnK_provider,
nu = (-2, +1, +1).
```

The conservation matrix contains independent hydrogen and oxygen rows with
columns `(H2O, H3O+, OH-)`:

```text
B = [[2, 3, 1],
     [1, 1, 1]].
```

Charge is enforced identically by the existing amount chart. The compiler must
verify `B nu^T = 0`, reaction charge conservation, ranks, feed neutrality,
Provider order/fingerprint/basis identity, IAPWS record completeness, reference
convergence, and reconstruction residual before Ipopt dispatch.

## Solve and evidence

The existing private fixed-`T,P` objective remains the sole Equilibrium owner:

```text
J(n,V) = Phi_Provider(T,n,V) + P V/(R T) + g_ref^T n.
```

The initial feed is liquid water with zero ionic feed amounts. Max-min
initialization must prove that a strictly positive conserved state exists; no
epsilon species is inserted. The existing exact electroneutral chart, direct
`lambda=1` attempt, adaptive continuation fallback, exact KKT assembly, and
postsolve certificates are reused without a second solver path.

Package tests cover bundle/source rejection, topology contributions, SDK tail
size and identity, reference convergence, analytic Provider derivatives,
standard-state sign and units, conservation/rank/charge, permutation and gauge
invariance, amount scaling, trace-floor sensitivity, false-success rejection,
and independent recomputation of the IAPWS activity product. A final isolated
cross-wheel test installs immutable Provider and Equilibrium wheels and proves
that no sibling source, retired lab runtime, or copied Provider equation is
loaded.
